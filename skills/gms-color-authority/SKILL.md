## Color Code Structure in SKUs

Every product ID encodes a color code. The position varies by product family, but the pattern is consistent:

```
Product ID: ANG114BK10
├── ANG   = Angle trim (family prefix, 2-4 chars)
├── 1     = gauge digit (not always present in all families)
├── 14    = profile size
├── BK    = Black (COLOR CODE, 1-4 chars)
└── 10    = 10 feet (length)

Color code position: after family prefix + gauge/size digits, before length digits
```

## Premium Finish Pricing

### The Three Premium Finishes

| Finish | Code | Markup | What It Looks Like |
|--------|------|--------|-------------------|
| **MCC** (Matte Color Coat) | MCC in SKU | **+15%** | Smooth, non-reflective matte surface. Modern, contemporary aesthetic. |
| **CR** (Crinkle) | C-prefix in color code | **+15%** | Textured wrinkled surface. Hides scratches and imperfections. Popular for agricultural. |
| **ULG** (Ultra Low Gloss) | ULG in SKU | **+15%** | Very subtle sheen, between matte and standard. Upscale understated look. |

The +15% markup applies to the ENTIRE product price, not just the material cost. A $50 standard panel becomes $57.50 in a premium finish.

### Finish Detection in Product IDs

Premium finishes are encoded in the color code portion of the SKU:
- **MCC**: Look for `MCC` in the color position (e.g., `ANG114MCC10`)
- **Crinkle**: Look for `C` prefix on color code (e.g., `CBK`, `CBS`, `CCH`, `CRR`)
- **ULG**: Look for `ULG` in the color position
- **Standard**: Any other color code = standard finish, no premium

## Common Mistakes

1. **Confusing SMP and Kynar versions of same color name** — "Ash Gray" exists in both systems. SMP Ash Gray (USS, $2.17 base) ≠ Kynar Ash Gray (CMG, $4.37 base). ALWAYS confirm paint system.
2. **Forgetting the +15% premium on crinkle/matte/ULG** — These are NOT just different colors. They carry a 15% markup on the entire product price.
3. **Quoting 26ga SMP colors that are "slitting" status** — Berry, Dark Red, Hawaiian Blue, etc. add 1-2 days processing on 26ga. Don't promise stock delivery.
4. **Assuming Kynar colors come in 29ga** — Kynar is CMG only: 24ga and 26ga. If they want 29ga, it's SMP only.
5. **Mixing up "Matte Black" paint vs "MCC" finish** — Matte Black (MBK) is a specific color. MCC (Matte Color Coat) is a finish type that can apply to multiple colors. Both carry premium pricing.
6. **Not asking "which tan?"** — There are 5+ tan-family colors (Tan, Saddle Tan, Buckskin, Taupe, Light Stone). Contractors say "tan" and mean different things.
7. **Ignoring solar reflectance for commercial** — Commercial projects often require ENERGY STAR compliance. Recommend lighter colors (SR ≥ 25%) for steep slope.
8. **Saying "120 SMP + 51 Kynar = 171 colors"** — Some color names overlap (Ash Gray, Charcoal Gray, Burnished Slate, Burgundy, Matte Black). The actual unique color count is 165+.
