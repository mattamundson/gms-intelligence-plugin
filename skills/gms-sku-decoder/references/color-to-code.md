## 3. Premium Finish Detection Quick Reference

### How to Identify Premium Finishes in a PID

Premium finishes appear **immediately after the gauge digit** in product SKUs:
- **Example:** `CO4129MCC10` = Coil, 40.875", **29ga (9)**, **Matte (MCC)**, 10'
- **Example:** `RC144CR10` = Ridge Cap, 14", **24ga (4)**, **Crinkle (CR)**, 10'

### USS (SMP Paint) Finishes

| Finish Code | Full Name | Gauge | Markup | Multiplier | Detection Pattern |
|-------------|-----------|-------|--------|-----------|------------------|
| MCC | Matte | 29, 26 | +8% | 1.08 | Code after gauge digit = "MCC" |
| CR | Crinkle | 29, 26 | +15% | 1.15 | Code after gauge digit = "CR" (NOT "CRRC" or "CRD") |
| (none) | Standard Gloss | 29, 26 | 0% | 1.00 | No finish code after gauge |

## 5. Quick Lookup Strategies

### If You Have a Product SKU:

1. **Extract the gauge digit:** Position after width code
   - 29ga = "9" or "29"
   - 26ga = "6" or "26"
   - 24ga = "4" or "24"

2. **Extract the color code:** Immediately after gauge
   - Check for premium finish: MCC, CR, ULG
   - Remove finish code to get base color

3. **Example:** `CO4129ARW10`
   - CO = Coil
   - 41 = Width (40.875")
   - 2 = Gauge shorthand (26ga)
   - 9 = Color lookup → ARW = Arctic White
   - (no premium) = Standard finish
   - 10 = Length (10')

### If You Have a Color Name:

1. **Find in Section 2 (Color Name → Code)**
2. **Note the supplier:** USS or CMG
3. **Check gauge compatibility:**
